NiceHash Buyer Profitability Chart

This website **estimates** the profitability of buying hashing power on NiceHash.

Calculations are made using my WIP nicehash-calculator rewrite (which uses WhatToMine). Fees are currently not accounted for.

View the source code. Report bugs or suggest features by making an issue.

To WhatToMine or NiceHash: If you would want me to reduce API requests, stop using your name, or anything like that then make an issue with the links above.

Yellow coins are the most profitable coin of their algorithm. The ROI column is the column you probably care the most about. (If you put Bitcoin in, you could expect to get this much of your bitcoin out)
